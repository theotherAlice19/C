## 1337 Piscine — C Programming

A 1337-day intensive coding bootcamp (Piscine) completed at 1337 School, covering C programming from the ground up. This repository contains all my work from C00 to C07, submitted daily across exams, rush projects, and exercises.

## What Projects I Build?

- **C00** — Basic I/O, writing functions from scratch, `write()`, first steps in C
- **C01** — Pointers, pointer arithmetic, passing by reference
- **C02** — String manipulation, `ft_strcpy`, `ft_strncpy`, character checks
- **C03** — String comparison and concatenation — `ft_strcmp`, `ft_strcat`, `ft_strncat`
- **C04** — Number conversion — `ft_strlen`, integer to string (`ft_itoa`), string to integer (`ft_atoi`)
- **C05** — Recursion — factorial, fibonacci, power, `sqrt`
- **C06** — `argc` and `argv` — working with command-line arguments
- **C07** — Dynamic memory allocation — `malloc`, `ft_strdup`, `ft_range`

## What I Practiced?

- Variables, data types, and operators in C
- Pointers and memory addresses
- String handling without the standard library
- Recursion and iterative logic
- Command-line arguments (`argc` / `argv`)
- Dynamic memory allocation with `malloc`
- Writing functions that mirror the C standard library (`libft` foundations)
- Working under the 42 Norm (code style rules)


## What is the Piscine?
The 1337 Piscine is a 4-week intensive selection process at 1337 School. It consists of:

- Daily C exercises submitted via git
- Weekly exams under timed conditions
- Weekend rush group projects
- Peer-to-peer evaluation

No teachers, no tutorials — only documentation, peers, and determination.
